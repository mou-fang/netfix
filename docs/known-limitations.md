# 已知限制

> 对应任务书 §0.2。记录当前不能检测或不能安全修复的问题。

## 阶段 0 限制

- **无任何诊断能力**：尚未实现任何探针，不能检测网络问题。
- **无任何修复能力**：尚未实现任何修复动作。
- **无历史记录**：尚未实现本地存储。
- **无报告导出**：尚未实现报告与脱敏。
- **无 Windows VM 测试**：无法进行故障注入测试。

## 后续阶段将持续更新的固有限制（任务书规定的安全边界）

- 不提供 VPN、代理节点或封锁绕过。
- 不自动卸载 VPN 驱动、虚拟网卡、安全软件。
- 不自动执行 Winsock/TCP-IP/全网络重置（只引导）。
- 不修改企业网络、组策略、静态 IP。
- 不登录或修改路由器后台。
- 不抓包、不做 HTTPS 中间人、不安装根证书。
- 不承诺清除病毒。
