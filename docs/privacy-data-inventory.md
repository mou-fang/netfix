# 隐私数据清单

> 对应任务书 §9.3。列出软件可能接触的隐私数据及处理策略。

## 数据分类与处理

| 数据类型 | 来源 | 默认处理 |
|---|---|---|
| Windows 用户名/用户目录 | 系统环境 | 报告中替换为占位符 |
| MAC 地址 | 网卡 API | 哈希或仅保留后四位 |
| SSID | WLAN API | 部分遮盖 |
| 公网/内网 IP | 网卡/连接 | 部分遮盖，保留类别和子网关系 |
| 企业域/DNS 后缀 | 系统配置 | 默认遮盖 |
| 代理 URL 凭据/Token/查询参数 | 代理配置 | 完全移除 |
| 应用完整路径 | 进程/文件选择 | 仅保留程序名（除非用户选完整报告） |
| hosts 条目 | hosts 文件 | 仅本次目标相关且脱敏 |
| DNS 缓存 | 系统 | 不导出无关域名 |
| 浏览/访问历史 | - | 不采集 |

## 原则

- 诊断数据默认只保存在本机（`%LocalAppData%/NetMedic/`）。
- 导出报告前显示预览。
- 不保存密码、Token、代理订阅内容。
- 无账号、无云端、无默认遥测。
